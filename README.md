# AgriBot: AI-Powered Crop Disease Prediction & Consultation

AgriBot is an intelligent, end-to-end solution designed to bridge the gap between advanced agricultural science and practical farming. By leveraging state-of-the-art **Generative AI** and **Computer Vision**, AgriBot identifies crop diseases through image analysis and provides actionable insights via a multi-language automated chatbot.

## 🚀 Project Overview
Developed as a final-year AI/ML project, AgriBot provides farmers with an accessible, real-time diagnostic tool. The application allows users to upload photos of distressed crops, receive an immediate diagnosis, and consult an AI assistant for treatment recommendations and preventative measures.

## ✨ Features
- **Disease Detection:** Utilizes high-performance image recognition to identify plant pathologies from uploaded images.
- **Automated Chatbot:** An integrated AI assistant to answer follow-up questions regarding soil health, fertilizers, and pest control.
- **Multi-language Support:** Designed for accessibility, allowing users to interact and receive reports in multiple local languages.
- **Real-time Analysis:** High-speed processing powered by the Google Gemini 1.5 Flash model for instant feedback.

## 🛠 Technology Stack
- **Frontend:** [Streamlit](https://streamlit.io/)
- **AI Engine:** [Google Generative AI](https://ai.google.dev/) (Gemini 1.5 Flash)
- **Image Processing:** [PIL](https://python-pillow.org/) (Python Imaging Library)
- **Language:** Python 3.10+
